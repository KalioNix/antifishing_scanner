크롬 확장 프로그램


누구나 간단하게 사용할 수 있는 안전한 사이트인지 더욱 쉽게 검사할 수 있는 크롬 확장프로그램 입니다.


DOWNLOAD

ABOUT 1.

  설치 방법  

- (1) : 다운로드 받은 확장 프로그램을 압축 해제 합니다.
- (2) : chrome://extensions/로 이동합니다.
- (3) : 우측 상단에 개발자 모드를 허용합니다.
- (4) : 압축 해제한 디렉터리를 업로드 합니다.
- (5) : 우측 상단 크롬 확장 프로그램에 추가됐는지 확인합니다.

ABOUT 2.

  사용 방법  

- 검사를 하고 싶을 때 우측 상단 크롬 확장 프로그램 "AFS"를 실행합니다.
- 검사 시작 버튼을 누르면, 검사하고자 하는 URL을 출력하고 해당 URL에 대한 신뢰 점수가 출력됩니다.

ABOUT 3.

  검사 결과  

<결과 그래프 색깔에 따른 안전도>
안전 : 안전한 사이트로 방문해도 좋습니다.
보통 : 비교적 안전한 사이트입니다.
주의 : 해당 웹사이트에서의 입력과 다운로드를 주의하세요.
위험 : 해당 사이트의 방문을 추천하지 않습니다.
