Main 사용법


증가하는 사기피해를 예방하기 위해 방문하는 웹 사이트를 점검할 방법이 필요하다고 느꼈으며,
개인도 보안에 신경 쓰도록 하는 인식 개선이 필요하다고 생각되어 개발하게 된 사이트입니다.

1. 메인 페이지

피싱검사 / 취약점검사 / 로그인

- URL 입력 창을 이용해 피싱검사와 취약점검사를 수행할 수 있습니다.
- 우측 상단에 로그인 버튼을 눌러 로그인하세요.

※ URL(인터넷주소) 입력시 http://www.naver.com과 같이
http, https를 붙여주세요!

2. 로그인

로그인(Login) / 회원가입(Signup)

Signup을 눌러 회원 가입 후 로그인하세요.
로그인 하면, 당신의 기록이 저장됩니다.

3. 피싱 검사 결과

url : 검사를 위해 입력한 인터넷 주소입니다.
IP Address : 검사를 위해 입력한 주소의 IP를 나타냅니다.
Score : 사이트가 얼마나 안전한지 정도를 점수로 나타냅니다.
Location : 검사를 위해 입력한 주소의 위치(나라)를 나타냅니다.
실행 시간 : 검사를 실행한 시간을 나타냅니다.
Open Port : 검사를 위해 입력한 주소의 서버에 열려 있는 포트번호를 나타냅니다.
KISA WHOIS : 한국인터넷진흥원에 등록 되어 있는 주소인지 나타냅니다.

<scoring 색깔에 따른 안전도>
안전 : 안전한 사이트로 방문해도 좋습니다.
보통 : 비교적 안전한 사이트입니다.
주의 : 해당 웹사이트에서의 입력과 다운로드를 주의하세요.
위험 : 해당 사이트의 방문을 추천하지 않습니다.

4. 취약점 검사 결과

Generate Report : 검사 결과를 편하게 저장하세요.
성공여부 : 항목에 대한 검사의 성공 여부가 나타납니다.
(성공 시 취약점이 발견된 것입니다.)
위험도 : 발견된 취약점의 위험도를 표시합니다.
공격 코드 : 발견된 취약점을 이용한 공격이 이루어 질 수 있는 코드입니다.
정보 : 눌러서 자세한 정보를 확인하세요.

<취약점 발견 개수에 따른 안전도>
안전 : 취약점이 거의 발견되지 않은 상태입니다. 발견됐더라도 심각하지 않습니다.
보통 : 위험도가 높은 취약점이 적게 발견되어도 보통 단계가 나올 수 있습니다. 위험도 낮은 취약점이 여러 개 발견되었더라도 보통 단계가 나올 수 있습니다.
주의 : 취약점이 많이 발견된 사이트입니다. 위험도가 높은 취약점이 발견됐을 가능성이 높습니다.
위험 : 취약점이 매우 많은 사이트입니다. 방문이나 이용에 주의가 필요합니다.
